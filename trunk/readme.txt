== Description ==

= Overview =
This plugin is for embedding a folders, photos or videos from the [Kuvat.fi](http://kuvat.fi) photo sharing service.

Kuvat.fi creates a great looking photo site from your photos. It's easy to use and you can use it with your web browser, tablets and mobile devices. We're based in Finland and have been running for 10 years.

= Usage example =
`[kuvat width=760 height=506 bgcolor=0 thumbstrip=0]http://demo.kuvat.fi/kuvat/[/kuvat]`

= Parameters =
*	**width**: The desired width of the embedded view. Required.
*	**height**: The desired height of the embedded view. Required.
*	**bgcolor**: This sets the background color and theme for the browser. Optional. Possible values:
	*	0: Dark gray (default)
	*	1: Black
	*	2: Light gray
*	**thumbstrip**: If set to 1, a film strip of the other photos in the same folder is shown. Optional. Possible values:
	*	0: Disabled (default)
	*	1: Enabled

== Installation ==

1. Upload `kuvatfi.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
